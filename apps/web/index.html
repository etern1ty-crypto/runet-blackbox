Open network observability

Понять, где ломается доступ

Runet Blackbox собирает безопасные отчёты о DNS, TCP, TLS и HTTP, чтобы отличать локальный сбой, деградацию провайдера, возможную блокировку и глобальную проблему сервиса.

English: privacy-first network measurement for unstable networks. Not a VPN, not a proxy, not a bypass tool.

git clone https://github.com/etern1ty-crypto/runet-blackbox.git

Не VPN Не проксирует трафик и не помогает обходить ограничения.
Без сервера Работает через локальный CLI, GitHub Issues и статический dashboard без собственного сервера.
Privacy-first Не публикует IP пользователя, headers, cookies, bodies, packet captures и точную геолокацию.
English-friendly Core docs and UI keep concise English context for international contributors.

Как пользоваться

Один запуск, один очищенный JSON, один GitHub issue

1Запусти одну цель или pack: --pack dev, --pack ai, --pack baseline.
2CLI создаст готовый Markdown для issue через --issue-file или попробует скопировать его через --copy-issue.
3Открой Measurement report issue и вставь готовый публичный отчёт.
4GitHub Actions повторно санитизирует отчёт и обновит агрегаты dashboard.

Что это не делает

Не обходит блокировки.

Не хранит сырые ответы сайтов.

Не собирает точную геолокацию.

Не заменяет юридическую или научную аттестацию измерений.
